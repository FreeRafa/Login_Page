# 🔐 Login Page (HTML + CSS)

## 📌 Descrição

Este projeto consiste na criação de uma página de login simples utilizando **HTML** e **CSS**, com foco em boas práticas de separação de responsabilidades (estrutura e estilo).

O objetivo principal é praticar:

* Estruturação de páginas com HTML
* Estilização com CSS
* Organização de arquivos
* Layout moderno com **Flexbox** e **Grid**

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

---

## 🎨 Funcionalidades

* Layout centralizado na tela
* Formulário de login com:

  * Campo de e-mail
  * Campo de senha
  * Botão de envio
* Efeitos de interação:

  * `hover` no botão
  * `focus` nos inputs
* Design simples e responsivo

---

## 📂 Estrutura do projeto

```
📁 login-page
 ├── index.html
 └── login.css
```

## 🧠 Aprendizados

Durante este projeto foram aplicados conceitos como:

* Separação entre HTML e CSS
* Uso de seletores CSS
* Layout com `flex` e `grid`
* Estilização de formulários
* Melhoria de experiência do utilizador (UX)
